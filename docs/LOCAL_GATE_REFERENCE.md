# Local Gate Reference

Use this page as a compact checklist for the local verification flow.

## Command

```powershell
npm run verify
```

## Expected finish

```text
FULL VERIFY GATE PASSED
```

## Follow-up check

```powershell
git status
```

Expected result:

```text
nothing to commit, working tree clean
```

For the longer operator workflow, read [`operator-runbook.md`](operator-runbook.md).
